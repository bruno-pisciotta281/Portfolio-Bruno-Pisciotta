Tecnologias Utilizadas

Uma ferramenta bem utilizada é um site bem construído

HTML 5

O HTML5 é uma linguagem de marcação responsável pela estrutura dos sites como um esqueleto, sem estilo e funçõesas construção.

CSS3

O CSS3 é uma linguagem de edentação responsável por estilizar e editar partes extremamene importantes do seu site e principalmente dar estilo.

JAVASCRIPT

O JAVASCRIPT é uma linguagem de programação que possui a função de dar funcionalidades a elementos principais dentro do site, entre outras funções.

BOOTSTRAP

O BOOTSTRAP é um FrameWork que possui o objetivo de facilitar o uso de alguns elementos dentro do site, unindo HTML, CSS e o JS em um mesmo conjunto de códigos.

Ao longo de minha tragetória na área da Tecnologia da Informação fui obtendo conhecimento destas tecnologias e ferramentas, estas das quais encremento em meus sites e softwares com o objetivo de tornar a experiência do usuário o mais satsfatória possível.
